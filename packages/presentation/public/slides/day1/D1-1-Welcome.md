## D1.1: Environment Check (Time: 15 mins)

**branch:** `day-1/00-welcome`

**Purpose:** Set expectations & confirm everyone is ready to code.

---

### Your Task:

1.  Open your terminal.
2.  Verify your setup:
    ```bash
    pnpm -v && node -v && git --version
    ```
3.  Install all project dependencies:
    ```bash
    pnpm install
    ```
4.  Start all development servers:
    ```bash
    pnpm dev
    ```
    _You should have three browser tabs open: the presentation, `vue-app`, and `react-app`._
